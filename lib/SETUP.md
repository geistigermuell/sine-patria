# Astronomy Engine Library

The solar system scene requires the `astronomy-engine` JavaScript library.

## Download

Run one of the following:

**Option A – npm / Node**
```
npm install astronomy-engine
cp node_modules/astronomy-engine/astronomy.browser.js lib/
```

**Option B – Direct download**
Download the file from:
https://cdn.jsdelivr.net/npm/astronomy-engine/astronomy.browser.js

Save it as: `Sine-Patria/lib/astronomy.browser.js`

Once the file is in place, reload Foundry VTT.
